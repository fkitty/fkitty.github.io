React Router

7月 05, 2020 发布在 React

什么是router

维基百科解释：路由（routing）就是通过互联的网络把信息从源地址传输到目的地址的活动
把一个地方的信息传到它想去的目的地的过程叫做路由

改变路由的实现方法

1. 使用hash做路由
2. 使用window.location.pathname
3. 使用window.histroy.pushState

三种方式的对比和选择：

  • window.location.hash (url变化，同时不刷新页面（选择）)
  • window.location.pathname （会刷新页面-基本不考虑）
  • window.histroy.pushState(1,2,3 ) （ 在不刷新页面的情况下改变pathname,是第二个的升级版（不确定：后端会将所有路径都指向首页的情况下可以做））

++那么问题来了，现在是两个页面，那页面变多的时候呢？3个页面，100个呢++

上述三方法的使用缺陷

  • 要写很多很长的代码去判断path
  • 当路由是不确定的时候,(path/000, path/111 …),无限多个的时候没办法去写
