Redux上篇-组件通信（EventHub)

8月 02, 2020 发布在 React

需求：实现如下图所示的组件结构之间任意组件之间进行通信

说明：每个组件都有一个共同的数据
分析：

  • 每个组件都有一个共同的数据，组件2在其内部改变数据之后，怎么通知组件3
  • 测试得知，当组件2改变数据之后，其他组件都不知道，因为它们没有在各自的内部setState
  • 组件2和组件3之间通信流程：C2->P1->App->P2->C3
  • 如果用上述流程，让任意组件之间通行，那这样做的事情就太多了

思路：

  • 假设有一个中间的传递者，让他和所有的组件交互

实现：

  • 经典的设计模式，发布订阅模式（eventHub）
